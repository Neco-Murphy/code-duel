## Usage
Grab a friend and head to [coralduel.azurewebsites.net](http://coralduel.azurewebsites.net/). Sign in, join a room, and do BATTLE.

## Requirements

- AngularJS 1.3.x
- Node 0.10.x
- Express 4.9.7
- Socket.io 1.0.x
